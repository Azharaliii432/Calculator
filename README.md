Calculator Web App

A simple calculator built using HTML, CSS, and JavaScript that performs basic arithmetic operations.

This project was created to practice DOM manipulation, event handling, and JavaScript logic without relying on libraries or frameworks.

Features

Addition, subtraction, multiplication, and division

Clear / reset functionality

Responsive and clean UI

Input handled through button clicks (no browser prompts)

Technologies Used

HTML – Structure of the calculator

CSS – Styling and layout

JavaScript – Core logic and calculations

What I Learned

Handling user input using JavaScript events

Updating the DOM dynamically

Writing clean and reusable JS logic

Separating structure, style, and behavior properly
